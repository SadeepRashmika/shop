- # Point of Sale (POS) System

## Project Overview

This project is a **Point of Sale (POS) System** that supports both **Sinhala and English languages**.  
The system includes three main user roles:

1. Owner  
2. Cashier  
3. Customer  

Users can switch between Sinhala and English at any time.

## 2. Cashier

The Cashier manages sales operations and customers.

### Item Management:
- Add, update, delete items
- Generate barcode for items
- Download or print barcodes
- Search items using:
  - Barcode
  - Item ID
  - Item Name

### Customer Management:
- Register new customers
- Update customer details
- Register debtors
- Generate barcode for debtors
- Search debtor details using barcode

### Billing:
- Create customer bills
- Print customer bills

### Reports:
- Generate daily sales report
- Generate monthly sales report
- Download reports

---

## 3. Customer

Customers have limited access.

### Customer Permissions:
- Create an account
- Login to the system
- Search items
- View item details
- Order items

Customers cannot:
- Manage items
- Access reports
- Manage other users
